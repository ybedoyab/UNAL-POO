# Ejercicio 6.7 – Validación de Campos

Documentación completa en `DOCUMENTATION.md`.

- Clase principal del enunciado: `unal.ejercicio6_7.EquipoMaratonProgramacion`.
- Clase auxiliar: `unal.ejercicio6_7.Programador`.
- Ejercicio propuesto integrado en el flujo principal: Validación de contraseñas del equipo.

## Ejecutar

```bash
mvn -q exec:java -Dexec.mainClass="unal.ejercicio6_7.Ejercicio6_7"
```

El programa ahora incluye:
- Registro de equipos de maratón de programación
- Validación de campos de nombres/apellidos
- **Validación de contraseñas del equipo** (integrada en el flujo principal)
- Control de capacidad del equipo (máximo 3 programadores)
